# Fairy_Puzzle

##Overview
iOS/Android向けパズルゲーム 

## Description
cocos2d-xフレームワークをを利用した落ちものパズルゲーム。
 現在iOSのみ対応

## Requirement
Xcode6.1
 cocos2d-x v3.3
 
## Usage
 スワイプで操作します。
 
## Install
 cocos newコマンドでプロジェクトを作成し、リソースとクラスを配置し、適宜プロジェクトでリンクを通してください。
